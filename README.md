# CreditWise Loan Prediction System

Machine Learning based loan approval prediction system developed using Logistic Regression, KNN, and Naive Bayes algorithms.

## Features
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Feature Engineering
- Multiple ML Models Comparison
- Streamlit Web Application Deployment

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- NumPy
- Streamlit

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes

## Deployment
Streamlit based interactive web application for real-time loan approval prediction.
